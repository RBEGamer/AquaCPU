# WaterGuard
A automatic aquarium refiller with extra fallout detection.



# PARTS
* 2 Water level sensors (reedswitch based or optical for example)
* RaspberryPI Zero Wifi + SD CARD
* magnet valve ( eg. 1/2" 10bar)
* Water flow sensor
* Rotary Encoder
* 16x2 I2c Display
* Arduino nano
* Current sensor
* GSM Module (eg SIM800L)
